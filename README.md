## How are graceful degredation and service workers related?
### Graceful degredation and service workers are related because service workers are used to enhance capable browsers, yet at the same time, older browsers don't lose an functionality. The only thing that they are missing is the offline caching benefits. The service worker improves performance and reliability but nothing gets broken on older browsers because that service worker registration code is just ignored. Modern browsers can still take advantage of service workers caching assets as you use them, apps working offline and with slow internet, and network requests are intercepted and served from cache when they are available.
